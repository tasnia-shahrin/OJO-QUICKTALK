# 📱 Ojo QuickTalk

**Ojo QuickTalk** is a real-time messaging application developed using **Java** and **Kotlin**, offering seamless and secure communication between users. Designed with responsiveness, performance, and user-friendliness in mind, this app supports real-time chat, notifications, and robust user management.



## 🚀 Features

- 🗨️ **Instant Messaging** – Real-time chat between users
- 🔒 **User Authentication** – Secure login and registration
- 👥 **User Presence** – See who's online/offline
- 📤 **Media Sharing** – Send images, files.
- 📱 **Android Support** – Native Android interface using Kotlin
- ☁️ **Backend in Java** – Scalable, secure, and efficient backend services

---

## ⚙️ Tech Stack

| Frontend (Mobile)      | Backend             | Real-time Engine  | Database        |
|------------------------|---------------------|-------------------|-----------------|
| Kotlin (Android)       | Java(Plain Java)    |  Firebase         | Firebase
